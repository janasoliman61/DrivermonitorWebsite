About this project

Driver Monitor Web is a small demo application that captures webcam frames in the browser and sends them to a backend for ML inference. The goal is to make it easy to plug in existing driver-monitoring models (drowsiness detection, smoking detection, phone-use detection, etc.) and view alerts in real time.

Who built this

Created by Jana. This repo is intended as a personal starter/experiment for building and iterating on driver-monitoring models and a minimal web UI.

Design goals

  • Simple and clear: minimal UI so the ML models are the focus.
  • Easy to extend: plug in TensorFlow / PyTorch / ONNX models on the server side.
  • Local-first: run locally for development and testing, then adapt for deployment.

Contact

If you want help integrating a model or adding features, update the repo or contact Jana.
